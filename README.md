### 安装方法

打开终端 ，输入：


```
git clone git@github.com:shenzekun/R-Todo.git
```

切换到克隆下来的文件夹：

```
cd R-Todo
```
然后输入：

```
npm install
```
开启：

```
npm start
```

### 实现功能
* 创建新账户
* 登录
* 注册
* 重置密码
* 表单验证
* 登出
* 使用 LeanCloud 存储所有数据
* Todo完成消息提示
* 修改 Todo
* 删除Todo
